把博客搬到ZEIT上

Posted on 2019-11-25 | Edited on 2021-04-16 | In 博客维护 | Views:
Symbols count in article: 156 | Reading time ≈ 1 mins.

一直用腾讯的开发者服务: coding 来发布网站，但从上周五开始，突然莫名其妙的不能更新了。

网页可以通过 git 更新到仓库中，但网页却始终不会改变，一直停留在上周五的页面，尝试和好几种方法去解决，但都不成功。最后决定尝试一下 ZEIT 这个服务。在全球有很多个CDN, 而且还有一个免费的区间。先试试，看看后续整体的体验。
