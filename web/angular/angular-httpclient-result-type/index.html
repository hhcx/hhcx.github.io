在使用 Angular 的 HttpClient 访问后台服务时，接收返回的Json值通常使用的是 any 类型，但使用这种方式往往会导致不能提前发现一些程序的问题，比如：名称拼写错误等…

解决的方法当然是避免使用 any, 而给返回值一个特定的类型。
