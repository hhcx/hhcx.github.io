升级Angular CLI

升级老项目的版本

如果Angular CLI的全局版本已经升级到最新版，但用老的Angular CLI生成的项目中运行相关命令(ng serve, ng build)时,会遇到一个警告，提示本地的版本与全局的版本不一致。这时，我们就可以考虑升级本地的版本。
