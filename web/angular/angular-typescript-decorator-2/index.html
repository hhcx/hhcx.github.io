在 上一篇文章 中，为大家介绍了 Typescript 中装饰器（Decorator）的概念及基本的声明方式。让读者对装饰器有一个基础的了解，本文将深入揭示装饰器的用法。

装饰器的分类

在 Typescript 中，装饰器按装饰的目标对象可分为：

  1. 类装饰器
  2. 方法装饰器
  3. 访问器装饰器
  4. 方法参数装饰器
  5. 属性装饰器

下面就为读者分别介绍这些装饰器的用法。
