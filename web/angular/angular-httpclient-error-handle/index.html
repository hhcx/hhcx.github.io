Angular 框架中内置了通过 HTTP/HTTPS 访问后端服务的模块 HttpClientModule, 通过模块中的 HttpClient 对象，开发者可以轻松的实现访问后端的 restful 接口。因为 HttpClient 采用了响应式的编程风格（rxjs)，所以很多初学者对于如何做错误处理会比较模糊，本文试图通过一些例子给读者一些指引。
