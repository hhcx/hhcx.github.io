Spring Boot 集成 Hibernate 中 ddl-auto 可取值

发表于 2020-10-12 | 更新于 2021-04-16 | 分类于 java ， SpringBoot
本文字数： 154 | 阅读时长 ≈ 1 分钟

Spring Boot 与 Hibernate 集成时，其 ddl-auto 中有以下一些可取的值:

update: 比较 entity 的信息与表结构，并将 entity 的更新应用到表结构中。
create: 如果表结构不存在则建立。
create-drop: 每次启动时重建表结构，并删除原有的数据。
none: 在启动时，不做任何处理。
