在Android开发中使用 Lombok 库

Posted on 2019-08-13 | Edited on 2021-04-16 | In App开发 , Android | Views:
Symbols count in article: 2.8k | Reading time ≈ 5 mins.

在企业 Android 应用开发中，往往需要面对大量的数据对象（POJO), 而且这些对象通常都会有比较的属性。为每个属性编写 get/set， toString 这些方法是件繁重而无趣的事情。使用 Lombok 库可以帮助我们解决这个问题。

安装 Lombok 插件

Lombok 是一种编译期代码生成技术，要想在 IDE 上能识别，需要安装对于的插件。在 Android Studio 中，安装步骤如下:

  1. 在菜单中选择 File -> Settings, 然后在弹出窗口中选择 Plugins
  2. 在弹出框的右边输入 Lombok
  3. 选择 Lombok 进行安装，如图：
  4. 安装完成以后，按照提示重启 Android Studio 即可

插件只需要安装一次，以后只需要在每个项目中设置 Lombok 支持就可以了。
