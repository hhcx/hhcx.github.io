微软认证的IoT开发板 - AZ3166的设置

Posted on 2019-11-26 | Edited on 2021-04-16 | In IoT技术 | Views:
Symbols count in article: 630 | Reading time ≈ 1 mins.

AZ3166是庆科信息和微软开发平台事业部联合研制的一套基于Azure平台的物联网开发套件。本文介绍一些该板的基本设置，以便于学员完成基于 Azure IoT 的相关课堂实验。

AZ3166简介

AZ3166开发板提供一个开箱即用的智能硬件解决方案，兼容Arduino平台，具有丰富的外围设备和传感器，包括:

  1. 一个温度和湿度传感器(HTS221)
  2. 一个气压传感器(LPS22HB)
  3. 一个动作传感器(LSM6DSL)
  4. 一个电磁感应器(LIS2MDL)
  5. 一个小屏幕的OLED
  6. 一个麦克风和一个耳机接口
  7. 一个红外接口
  8. 一个三色LED灯

如图:

因此，该版适用于物联网、智能硬件的原型机开发，方便验证用户的软件和功能，使产品可以快速、安全地连接至Azure云服务平台和手机端，缩短研发周期。

进入AP模式

AP模式是为了帮助AZ3166连入本地无线网络的必要步骤，在该模式中，AZ3166将作为一个无线AP来使用，地址为 192.168.0.1, 用电脑或手机连上网以后，访问

就可以看到输入SSID和密码的界面，这样就可以为AZ3166配置无线网络了。

要进入AP模式，执行以下步骤:

  1. 按住B键不放
  2. 按下Reset键
  3. 放开Reset键
  4. 放开B键

这是屏幕上将显示进入 AP模式 的字样。

进入配置模式

对于 AZ3166 ，当要配置 链接串是，需要进入配置模式，执行以下步骤:

  1. 按住A键不放
  2. 按下Reset键
  3. 放开Reset键
  4. 放开A键

这是屏幕上将显示进入 配置模式(Configuration) 的字样。
