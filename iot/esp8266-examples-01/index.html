作为入门，当然是永远的 hello world。

SDK 中自带的 hello world 例程位于 examples/get-started/hello_world 目录中。 项目包含一个 C 语言的源文件 - hello_world_main.c（位于 main 目录中), 其它的文件是 Make 构建系统需要的配置文件。
