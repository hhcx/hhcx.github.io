通过对 DataFrame 合并规则的分析可以知道， Merge操作是需要有列名间存在对应关系为前提的，但我们通过 value_counts 函数最终生成的序列 （Series) 实际上是没有列名的，应该Merge函数也没法找到合并的条件。解决的方法就是为序列 (Series) 指定列名和索引名，注意不用忘了给索引名。其实每个序列 (Series) 都可以看作是一个只有两列的二维表，一列是索引，一列是数据。
